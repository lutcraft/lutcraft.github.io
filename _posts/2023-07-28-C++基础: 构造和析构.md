## 构造函数

与类名完全相同的成员函数为这个类的构造函数,构造函数没有返回值.

```cpp
class A
{
    int i;
    A();
}

A::A(int i)
{
    this.i = i;
}

```


## 析构函数
析构函数的命名为在构造函数名前加~,注意析构函数不能有参数.
```cpp
class A
{
    int i;
    A();
    ~A();
}

A::A(int i)
{
    this.i = i;
}
A::~A()
{
    
}
```


## 何时构造和析构

在对象创建时创建调用构造函数,在对象的作用域终结时,析构.


举例:

```cpp
int main()
{
    {
        A a;        //构造
        a.i = 1;
    }               //析构
}                   //注意:不是在这里析构的!

```
